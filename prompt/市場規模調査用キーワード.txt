
市場規模調査用キーワード

修正後の指示文例

指示内容:
	1.	ユーザーから与えられた事業内容をもとに、関連キーワードを以下の3つのリストに分割して返してください。
	•	coreKeywords: 「事業の主要な対象や領域」に関するキーワード。ユーザーから提供された情報に限らず、事業の意図や目的を基にAIが関連するコアなキーワードを生成・補完してください。
	•	actionKeywords: 「回収・再利用など行為や動作」に関するキーワード
	•	relatedKeywords: 「事業やサービス、技術内容に直接関連する具体的な名詞」に関するキーワード。実装方法や詳細な機能など、事業のコアや直接の関連性が低いキーワードは含めないでください。
	2.	各リスト内のキーワードは関連度の高い順に並べてください。
	•	できるだけ多くのキーワードを出してください。
	•	重複するキーワードは排除してください。
	3.	キーワードはできるだけ「単語や短いフレーズ」で格納してください。
	•	例: "レアメタル回収" → "レアメタル" と "回収" のように分ける。
	4.	actionKeywordsリストでは、以下を遵守してください:
	•	動詞や名詞としての行為を中心にキーワードを選定してください。
	•	不要な修飾語や冗長な表現（例: “広範囲回収”）は避け、基本的な動作のみを含めてください。
	5.	relatedKeywordsリストでは、以下を遵守してください:
	•	事業やサービス、技術内容に直接関連する具体的な名詞のみを含めてください。
	•	抽象的な形容詞や一般的な用語（例: “高効率”, “安定供給”, “環境負荷軽減”）は含めないでください。
	6.	**出力は以下のJSON形式で返してください。JSONデータのみを出力し、その他のテキストやコメントは含めないでください。**
	
{
  "coreKeywords": [
    // ここに「事業の主要な対象や領域」に関するキーワード
  ],
  "actionKeywords": [
    // ここに「回収やリサイクル」など行動・動作に関するキーワード
  ],
  "relatedKeywords": [
    // ここに「事業やサービス、技術内容に直接関連する具体的な名詞」などのキーワード
  ]
}

目的:
	•	事業内容に基づき、各種キーワードの「AND検索」や「組み合わせ検索」を行うため。
	•	カテゴリーごとに分かれていることで、coreKeywords と actionKeywords を AND検索 で組み合わせて最適な市場やマーケットを見つけられるようにする。
	•	組み合わせ検索で結果が得られない場合は、relatedKeywords のみで検索し、さらに結果が得られない場合は、coreKeywords のみで検索することで、最適な検索結果を得る。

具体的な指示内容のポイント:
	1.	リストの明確な分類:
	•	coreKeywords: 事業が対象とする主要な対象や領域に関連するキーワードを含めます。ユーザーから提供された情報に限らず、事業の意図や目的を基にAIが関連するコアなキーワードを生成・補完してください。
	•	actionKeywords: 実際の行為や動作に関連するキーワードを含めます。
	•	relatedKeywords: 事業やサービス、技術内容に直接関連する具体的な名詞を含めます。実装方法や詳細な機能など、事業のコアや直接の関連性が低いキーワードは含めないでください。
	2.	重複の排除:
	•	各リスト内で同一または類似のキーワードが重複しないように注意します。
	3.	不要な修飾語の除去:
	•	actionKeywordsでは、動作の本質を表すキーワードに集中し、不要な修飾語を避けます。
	4.	具体性の確保:
	•	relatedKeywordsには事業やサービス、技術内容に直接関連する具体的な名詞のみを含め、抽象的なキーワードは排除します。
	•	**AIはユーザーから提供された文章を基に、事業の意図や目的を深く理解し、それに基づいて関連するキーワードを生成してください。単なるキーワードの抽出にとどまらず、事業の核心に触れるキーワードを積極的に含めるようにしてください。**

期待される出力例:

{
  "coreKeywords": [
    "起業家適性",
    "起業支援",
    "起業家精神",
    "新規事業創出",
    "キャリア支援",
    "性格適正"
  ],
  "actionKeywords": [
    "診断",
    "分析",
    "提供",
    "支援"
  ],
  "relatedKeywords": [
    "診断システム",
    "教育機関"
  ]
}

注意点:
	•	actionKeywordsから「広範囲回収」を除外し、「コスト抑制」の重複も解消しました。
	•	relatedKeywordsには事業やサービス、技術内容に直接関連する具体的な名詞のみを含め、実装方法や詳細な機能は含めないようにしました。
	•	各リストは関連度の高い順に並べ、重複や不要な修飾語を含まないようにしています。
	•	relatedKeywordsがユーザーの入力に含まれていない場合でも、AIが関連する具体的なキーワードを生成して埋めるようにしてください。
	•	**AIはユーザーから提供された文章を基に、事業の意図や目的を深く理解し、それに基づいて関連するキーワードを生成してください。単なるキーワードの抽出にとどまらず、事業の核心に触れるキーワードを積極的に含めるようにしてください。**

検索の運用方法:
	1.	主検索:
	•	coreKeywords と actionKeywords の各リストからキーワードを1つずつ選び、AND演算で検索します。
	•	例: "起業家適性" AND "診断"
	2.	フォールバック検索:
	•	主検索で結果が得られなかった場合、relatedKeywords のみで検索します。
	•	例: "診断システム"
	3.	最終検索:
	•	それでも結果が得られない場合、coreKeywords のみで検索します。
	•	例: "起業家適性"

目的:
	•	複数のキーワードリストを効果的に組み合わせることで、最適な市場やマーケットを効率的に特定する。
	•	フォールバック検索を活用することで、検索範囲を広げつつも精度を維持する。

この指示文により、AIはユーザーの意図に沿った、整理されたキーワードリストをJSON形式で提供できます。特に、coreKeywords と actionKeywords を組み合わせたAND検索を基本とし、必要に応じて relatedKeywords を活用することで、柔軟かつ精度の高い市場調査が可能となります。さらに、relatedKeywordsが不足している場合でも、AIが適切な具体的キーワードを生成することで、検索の精度を向上させます。
